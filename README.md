LinkTree-landingPage

Uma landing page estilo LinkTree, criada com HTML, CSS e JavaScript. O objetivo desse projeto é oferecer uma página de “link-in-bio” personalizada, leve e responsiva, que pode ser usada para centralizar seus links e informações pessoais ou profissionais.

 Demonstração

[ [ ](https://barbara-ellen-linktree.vercel.app/)](https://barbara-ellen-linktree.vercel.app/)


  Sobre o Projeto

Esse projeto foi desenvolvido como uma página de links simples e customizável, inspirada no conceito do LinkTree. Ele é ideal para usar em redes sociais, currículo ou portfólio, mostrando seus principais links e apresentando o trabalho de forma visualmente agradável e organizada.

Principais características:

 Responsivo — funciona bem em dispositivos móveis e desktops

 Estrutura simples com HTML, CSS e JavaScript

 Possibilidade de personalização visual e de links

 Código organizado por páginas e estilos


 Tecnologias Utilizadas

Esse projeto foi feito com:

HTML5

CSS3

JavaScript (simples, sem frameworks)

 Como Executar Localmente

Clone este repositório:

git clone https://github.com/carlacavalcanti/LinkTree-landingPage.git

Entre na pasta:

cd LinkTree-landingPage

Abra o arquivo index.html no seu navegador.

 Não é necessário instalar dependências, pois é um projeto estático.
